# About Shopify Plugin
> Shopify plugin 是一套基于NexAmerchant 开发与设计的 多Shopify 店铺管理运营的平台,基于Shopify Admin API 接口，实现对多Shopify店铺的管理运营，从而更好的提升Shopify店铺的效率
